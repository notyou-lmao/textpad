# Textpad Editor
Creates a moveable textpad on screen.

---

### Features
- Undo/redo
- Run button (runs script in chrome dev console)
- Save+load (saves and loads documents to and from your files)
- Line wrap toggle
- Minimize button
- Close button (deletes the pads html from the dom)
- Resizeable
